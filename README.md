# üéµ Analyrics - Frontend

Analyrics is a modern, interactive web application designed to analyze music lyrics. It provides users with deep insights into their favorite songs, including "vibe" checks, core message extraction, and slang/metaphor decoding, all presented in a visually stunning interface.s

## ‚ú® Features

-   **Deep Lyric Analysis**: powered by AI to understand the hidden meanings, metaphors, and cultural context of songs.
-   **Vibe Check**: Instantly get the mood and atmosphere of a track.
-   **Slang & Metaphor Decoder**: Break down complex phrases and urban slang into plain language.
-   **Interactive Dashboard**: A sleek, card-based interface to manage your recent analyses.
-   **Immersive UI/UX**:
    -   Smooth animations with **Framer Motion**.
    -   Glassmorphism and modern aesthetics using **TailwindCSS**.
    -   Responsive design for all devices.

## üöÄ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### Prerequisites

-   Node.js (v18 or newer recommended)
-   npm, yarn, or bun

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/DatTruong0108/analyrics-fe.git
    cd analyrics-fe
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    yarn
    # or
    bun install
    ```

3.  **Configure Environment**:
    Create a `.env` file in the root directory if necessary (check with the backend team for API endpoints).
    ```env
    NEXT_PUBLIC_API_URL=
    ```

4.  **Run the development server**:
    ```bash
    npm run dev
    ```

5.  **Open your browser**:
    Navigate to [http://localhost:3000](http://localhost:3000) to see the app in action.
